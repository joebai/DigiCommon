Setup
=====

Add the following to your composer.json file inside your project root.

```
{
        "repositories":[
                {
                    "type": "vcs",
                    "url": "https://github.com/Tyler-Gauch/DigiCommon.git"
                }
            ],
            "require": {
                ....,
                "digi/common": "dev-master"
            }
}
```
